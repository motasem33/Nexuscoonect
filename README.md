NexusConnect 

A high-performance Flutter mobile application designed to showcase clean architecture and efficient state management. Built as a demonstration for API consumption, focusing on modern UI/UX principles and enterprise-ready coding standards.

 Key Features
- **Clean Architecture:** Separated concerns between UI, Business Logic (Provider), and Data layers.
- **State Management:** Implemented using `provider` for efficient reactivity.
- **RESTful API Integration:** Optimized HTTP client with robust error handling for JSONPlaceholder.
- **Modern UI:** Responsive design using Inter font, custom themes, and Shimmer loading effects for improved UX.

Tech Stack
- **Framework:** Flutter & Dart
- **State Management:** Provider
- **Networking:** HTTP
- **Architecture:** MVVM/Feature-first approach
